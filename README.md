# BeRich

个人脚本仓库，包含多个自动化脚本和配置模块。

## 项目列表

### 1. [ALIYUN (阿里云)](./ALIYUN)

阿里云社区自动任务脚本，支持 Surge。
- 自动完成签到、点赞、收藏、评论等任务
- 积分自动领取
- 库存查询

### 2. [weibo (微博)](./weibo)

新浪微博每日自动签到脚本。
- 支持多账号
- 自动领取签到奖励

### 3. [JDCK (京东 Cookie 同步)](./JDCK)

自动捕获京东 Cookie 并同步到青龙面板。
- 支持 Surge 和 Loon
- 自动更新 Cookie

### 4. [NFSQ (农夫山泉)](./NFSQ)

农夫山泉小程序自动抽奖脚本。
- 自动完成每日任务
- 双通道混合抽奖
- 自动领取奖品

## 使用说明

请进入各个项目的子目录查看详细的 `README.md` 使用文档。
