# BeRich 🚀

个人自动化脚本合集，旨在让生活更智能、更高效。包含针对 Surge 和 Loon 的多项优质脚本。

---

## 📂 项目列表

| 项目 | 描述 | 支持平台 |
| :--- | :--- | :--- |
| [**weibo (微博)**](./weibo) | 微博每日自动签到，支持多账号。 | Surge, Loon |
| [**JDCK (京东)**](./JDCK) | 自动化。自动捕获京东 Cookie 并同步至青龙面板。 | Surge, Loon |
| [**NFSQ (农夫山泉)**](./NFSQ) | 农夫山泉小程序自动任务与抽奖。 | Surge, Loon |

---

## 🛠️ 通用使用指南

1.  **环境准备**: 各项目均基于脚本工具（Surge 或 Loon）运行，请确保您已安装并激活。
2.  **证书配置**: 必须开启 **MITM** 功能并正确安装/信任证书，否则无法捕获 Cookie 或 Token。
3.  **模块安装**: 建议优先使用模块（`.sgmodule`）或插件（`.plugin`）进行一键安装。
